    CREATE backend/app/schemas/cost.py with these Pydantic models (no `from_attributes=True` — these are plain BaseModel, not ORM models; results come from result.all() named tuples, not ORM instances):

    ```python
    from pydantic import BaseModel
    from typing import Optional

    class SpendSummaryResponse(BaseModel):
        mtd_total: float
        projected_month_end: float
        prior_month_total: float
        mom_delta_pct: Optional[float]  # null when prior month has zero spend (first period)

    class DailySpendResponse(BaseModel):
        usage_date: str   # ISO date string, e.g. "2026-01-15"
        total_cost: float

    class BreakdownItemResponse(BaseModel):
        dimension_value: str
        total_cost: float

    class TopResourceResponse(BaseModel):
        resource_id: str
        resource_name: str
        service_name: str
        resource_group: str
        total_cost: float
    ```

    CREATE backend/app/services/cost.py with these five async functions:

    1. `get_spend_summary(session: AsyncSession) -> dict` — computes MTD total, projection, prior month total, MoM delta:
       - MTD: `SELECT SUM(pre_tax_cost) WHERE YEAR(usage_date)=today.year AND MONTH(usage_date)=today.month`
       - Use `extract("year", BillingRecord.usage_date)` and `extract("month", ...)` for SQLAlchemy async
       - Projection formula: `(mtd / days_elapsed) * days_in_month` using `calendar.monthrange`
       - Prior month: subtract one month from today (handle January → December prior year)
       - MoM delta: `((mtd - prior) / prior * 100)` if prior > 0 else None
       - Cast all Decimal results to float: `float(result.scalar() or 0.0)`

    2. `get_daily_spend(session: AsyncSession, days: int) -> list` — GROUP BY usage_date SUM:
       - `WHERE usage_date >= date.today() - timedelta(days=days)`
       - `ORDER BY usage_date ASC`
       - Returns `result.all()` (NOT `.scalars().all()` — multi-column query)
       - Pass `connectNulls=True` note: the service fills no gaps — frontend uses recharts `connectNulls`

    3. `get_breakdown(session: AsyncSession, dimension: str, days: int) -> list` — dynamic GROUP BY:
       ```python
       DIMENSION_MAP = {
           "service_name": BillingRecord.service_name,
           "resource_group": BillingRecord.resource_group,
           "region": BillingRecord.region,
           "tag": BillingRecord.tag,
       }
       ```
       Raises `ValueError("Invalid dimension: {dimension}")` if key not in map.
       `WHERE usage_date >= cutoff, GROUP BY col, ORDER BY SUM(pre_tax_cost) DESC`

    4. `get_top_resources(session: AsyncSession, days: int) -> list` — top 10 by total cost:
       ```python
       stmt = (
           select(
               BillingRecord.resource_id,
               BillingRecord.resource_name,
               BillingRecord.service_name,
               BillingRecord.resource_group,
               func.sum(BillingRecord.pre_tax_cost).label("total_cost"),
           )
           .where(BillingRecord.usage_date >= cutoff)
           .group_by(
               BillingRecord.resource_id,
               BillingRecord.resource_name,
               BillingRecord.service_name,
               BillingRecord.resource_group,
           )
           .order_by(func.sum(BillingRecord.pre_tax_cost).desc())
           .limit(10)
       )
       ```
       Filter out rows where resource_id is empty string: `.where(BillingRecord.resource_id != "")` — avoids showing "Untagged" bucket at top.

    5. `get_breakdown_for_export(session: AsyncSession, dimension: str, days: int) -> list` — same as get_breakdown but no LIMIT, used by CSV export endpoint. Reuse DIMENSION_MAP, same query, no limit.

    Follow established project pattern: import `get_settings` at function-call time if settings are needed (they are not for cost service — only DB session is needed). Use `from app.core.database import AsyncSessionLocal` is NOT needed here — session is injected via FastAPI DI. Import `from sqlalchemy.ext.asyncio import AsyncSession`.

    All functions use `result.all()` for multi-column queries. Map to floats at the API layer, not the service layer (service returns raw rows).

    ```python
    import csv
    import io
    from fastapi import APIRouter, Depends, Query, HTTPException
    from fastapi.responses import StreamingResponse
    from sqlalchemy.ext.asyncio import AsyncSession
    from app.core.dependencies import get_db, get_current_user
    from app.models.user import User
    from app.schemas.cost import (
        SpendSummaryResponse, DailySpendResponse,
        BreakdownItemResponse, TopResourceResponse,
    )
    from app.services.cost import (
        get_spend_summary, get_daily_spend,
        get_breakdown, get_top_resources, get_breakdown_for_export,
    )

    router = APIRouter(prefix="/costs", tags=["costs"])
    ```

    Implement these 6 endpoints:

    1. `GET /costs/summary` → `SpendSummaryResponse`:
       ```python
       @router.get("/summary", response_model=SpendSummaryResponse)
       async def spend_summary(
           db: AsyncSession = Depends(get_db),
           _: User = Depends(get_current_user),
       ):
           data = await get_spend_summary(db)
           return SpendSummaryResponse(**data)
       ```

    2. `GET /costs/trend?days=30` → `list[DailySpendResponse]`:
       Validate `days` with `Query(default=30, ge=1, le=90)`. Map rows:
       `[DailySpendResponse(usage_date=str(r.usage_date), total_cost=float(r.total_cost)) for r in rows]`

    3. `GET /costs/breakdown?dimension=service_name&days=30` → `list[BreakdownItemResponse]`:
       Validate `dimension` with `Query(default="service_name", pattern="^(service_name|resource_group|region|tag)$")`.
       Catch `ValueError` from service layer and raise `HTTPException(status_code=400, detail=str(e))`.
       Map: `[BreakdownItemResponse(dimension_value=str(r.dimension_value or ""), total_cost=float(r.total_cost)) for r in rows]`
       Filter out rows with empty dimension_value from the response (they'd show as blank rows in UI).

    4. `GET /costs/top-resources?days=30` → `list[TopResourceResponse]`:
       Validate `days` with `Query(default=30, ge=1, le=90)`.
       Map: `[TopResourceResponse(resource_id=r.resource_id, resource_name=r.resource_name, service_name=r.service_name, resource_group=r.resource_group, total_cost=float(r.total_cost)) for r in rows]`

    5. `GET /costs/export?dimension=service_name&days=30` → CSV StreamingResponse:
       ```python
       @router.get("/export")
       async def export_costs(
           dimension: str = Query(default="service_name", pattern="^(service_name|resource_group|region|tag)$"),
           days: int = Query(default=30, ge=1, le=90),
           db: AsyncSession = Depends(get_db),
           _: User = Depends(get_current_user),
       ):
           rows = await get_breakdown_for_export(db, dimension=dimension, days=days)
           output = io.StringIO()
           writer = csv.writer(output)
           writer.writerow(["dimension", "total_cost_usd"])
           for row in rows:
               writer.writerow([str(row.dimension_value or ""), float(row.total_cost)])
           output.seek(0)  # CRITICAL: reset buffer AFTER writing, BEFORE streaming
           return StreamingResponse(
               iter([output.getvalue()]),
               media_type="text/csv",
               headers={"Content-Disposition": "attachment; filename=cost-breakdown.csv"},
           )
       ```

**utcnow() helper:** Redefine locally (same pattern as billing.py and recommendation.py — do NOT import from another model file).

```python
from datetime import datetime, date, timezone
def utcnow() -> datetime:
    return datetime.now(timezone.utc)
```

**Model 1: TenantProfile**
- `__tablename__ = "tenant_profiles"`
- `id`: UUID primary key (uuid.uuid4 default)
- `tenant_id`: String(255), NOT NULL, unique=True — raw tag value from BillingRecord.tag
- `display_name`: String(255), nullable=True — admin-set human-readable name
- `is_new`: Boolean, NOT NULL, default=True — drives "New" badge in UI
- `acknowledged_at`: DateTime(timezone=True), nullable=True
- `first_seen`: Date, NOT NULL
- `created_at`: DateTime(timezone=True), NOT NULL, default=utcnow
- `updated_at`: DateTime(timezone=True), NOT NULL, default=utcnow, onupdate=utcnow

**Model 2: AllocationRule**
- `__tablename__ = "allocation_rules"`
- `id`: UUID primary key (uuid.uuid4 default)
- `priority`: Integer, NOT NULL — first-rule-wins ordering
- `target_type`: String(50), NOT NULL — 'resource_group' or 'service_category'
- `target_value`: String(255), NOT NULL — e.g. "rg-shared" or "Compute"
- `method`: String(50), NOT NULL — 'by_count', 'by_usage', or 'manual_pct'
- `manual_pct`: JSON, nullable=True — `{tenant_id: pct}` dict, only used when method='manual_pct'
- `created_at`: DateTime(timezone=True), NOT NULL, default=utcnow
- `updated_at`: DateTime(timezone=True), NOT NULL, default=utcnow, onupdate=utcnow
- `__table_args__`: UniqueConstraint("priority", name="uq_allocation_rule_priority")

**Model 3: TenantAttribution**
- `__tablename__ = "tenant_attributions"`
- `id`: UUID primary key (uuid.uuid4 default)
- `tenant_id`: String(255), NOT NULL — 'UNALLOCATED' sentinel for unmatched costs; never corresponds to a TenantProfile
- `year`: Integer, NOT NULL
- `month`: Integer, NOT NULL
- `total_cost`: Numeric(18, 6), NOT NULL, default=0
- `pct_of_total`: Numeric(10, 4), NOT NULL, default=0
- `mom_delta_usd`: Numeric(18, 6), nullable=True — None if no prior month data
- `top_service_category`: String(255), nullable=True
- `allocated_cost`: Numeric(18, 6), NOT NULL, default=0 — cost from allocation rules
- `tagged_cost`: Numeric(18, 6), NOT NULL, default=0 — directly tagged cost
- `computed_at`: DateTime(timezone=True), NOT NULL, default=utcnow
- `updated_at`: DateTime(timezone=True), NOT NULL, default=utcnow, onupdate=utcnow
- `__table_args__`:
  - UniqueConstraint("tenant_id", "year", "month", name="uq_tenant_attribution_key")
  - Index("idx_attribution_year_month", "year", "month")
  - Index("idx_attribution_tenant_id", "tenant_id")
